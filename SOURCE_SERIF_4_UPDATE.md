# Source Serif 4 Reading Font Update

- Normal content inside Notes, Summary, and Q&A now uses **Source Serif 4**.
- Reader headings and topic headings continue to use **Fraunces**.
- Physics formulas, chemistry equations, reactions, ions, and other detected scientific expressions continue to use the **Cambria Math** font stack.
- The service-worker cache was bumped to `ncert-study-archive-v6` so deployed clients receive the updated styles.
