# KaTeX equation update

- Added KaTeX 0.16.9 and the auto-render extension.
- Added the KaTeX mhchem extension for chemistry reactions.
- Physics and chemistry equations are detected and rendered with textbook-quality mathematical typography.
- Common Unicode superscripts, subscripts, Greek symbols, powers, arrows and simple fractions are converted for KaTeX.
- Notes no longer use a chat conversation layout. Each topic name appears first, followed directly by its content and points.
- Updated the service-worker cache version so deployed users receive the new files.
