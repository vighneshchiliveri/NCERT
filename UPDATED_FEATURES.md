# Updated website features

This version keeps the site fully static and ready for Vercel.

## Added

- Global chapter search in the header (`Ctrl/Cmd + K`)
- Shareable URLs for class, subject, textbook, chapter and content type
- Browser Back/Forward navigation support
- Clickable breadcrumbs
- Notes/Summary/Q&A availability badges on chapter cards
- Reader tabs for switching content without leaving the page
- Font-size, line-spacing and paper/sepia/dark reader controls
- Bookmarks, completed chapters and Continue Learning using local browser storage
- Reading-time estimate
- Share button with native share or copy-link fallback
- Improved PDF output with page numbers; Print → Save as PDF fallback for Unicode text
- Responsive mobile reader controls
- Installable PWA and offline caching for previously opened content
- Improved accessibility labels, focus handling and reduced-motion support
- Independent-resource disclaimer and issue-report link

## Deployment

Upload the full folder to GitHub or Vercel exactly as before. There is no build command.
